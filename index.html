LES SOURIS

J'utilise actuellement la Zowie EC2A qui est une souris pour les droitiers uniquement, vous pouvez changer la taille si vous voulez une souris plus grosse (EC1) directement sur le lien amazon. Concernant la FK1 c'est une souris de chez Zowie aussi mais cette fois ambidextre, la Logitech GPRO est la meilleure souris sans-fil sur le marché.
